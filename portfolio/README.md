# XPro-Servis Portfolio

Современный портфолио веб-разработчика с блогом.

## Технологии

- **Next.js 16** - React фреймворк
- **React 19** - Компоненты
- **TypeScript** - Типизация
- **Tailwind CSS 4** - Стили
- **Framer Motion** - Анимации

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Сборка

```bash
npm run build
npm start
```

# Быстрое решение проблемы DEPLOYMENT_NOT_FOUND

## Три простых шага:

### 1️⃣ Войдите в Vercel
```powershell
vercel login
```
*(Откроется браузер для входа)*

### 2️⃣ Свяжите проект
```powershell
vercel link
```
*(Следуйте инструкциям в терминале)*

### 3️⃣ Задеплойте
```powershell
vercel --prod
```

**Готово!** Ваш сайт будет доступен по URL, который покажет Vercel.
